建模背景：
在构建定量分析模型时，线性关系是最基础且广泛应用的数学表达形式。该模型假设输出变
量与输入变量之间存在线性相关性，即输出变量随输入变量按固定比例变化，并包含一个固
定的偏移量。此类模型广泛应用于回归分析、系统建模以及机器学习的初期理论构建中。通
过设定权重和偏置项，可以对输入数据进行线性变换，以预测输出结果。
本模型旨在模拟一个单变量线性函数，用于描述输入值经过线性运算后所对应的输出值，适
用于基础教学、算法验证以及模型初始化等场景。

其中：
• $ x $ 表示输入变量；
• $ w $ 表示权重系数，反映输入对输出的影响程度；
• $ b $ 表示偏置项，代表输出的基准值；
• $ y $ 表示模型计算得到的输出结果。

```
@mcp.tool
def linear_model(
    x: Annotated[float, "输入变量"],
    w: Annotated[float, "权重系数"],
    b: Annotated[float, "偏置项"]
):
    """单变量线性模型，用于计算给定输入x通过权重w和偏置b后的输出y"""
    import sympy as sp
    
    # 定义符号变量
    x_sym, w_sym, b_sym = sp.symbols('x_sym w_sym b_sym')
    
    # 构建公式
    y_sym = w_sym * x_sym + b_sym
    
    # 代入参数计算结果
    y = y_sym.subs({x_sym: x, w_sym: w, b_sym: b})
    
    return float(y)
```
