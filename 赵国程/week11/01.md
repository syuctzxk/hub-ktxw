1. 知识库文件上传
- Path: /document/upload
- Method: POST
- RequestParam: 
  - knowledge_id(str): 知识库id
  - name(str): 文档名称
  - file(file): 要上传的文件
  - type(str): 要上传的文件类型, 可选值: pdf, doc
- ResponseBody:
  ```
  {
    "code": 200,
    "msg": "success"
  }
  ``
2. 知识库文件删除

- Method: POST
- RequestParam: 
  - knowledge_id(str): 知识库id
- ResponseBody:
  ```
  {
    "code": 200,

4. 知识库文件查询
- Path: /document/query
- Method: GET
- RequestParam: 
  - knowledge_id(str): 知识库id
  - page(int): 页码, 从1开始
  - page_size(int): 每页数量,
- ResponseBody:
  ```
  {
    "code": 200,
    "msg": "success",
    "data": [
      {
        "id": "1",
        "name": "文档1",
        "type": "pdf",
        "url": "https://example.com/document1.pdf"
      },
      {
        "id": "2",
        "name": "文档2",
        "type": "doc",
        "url": "https://example.com/document2.doc"
      }
    ]
  }
  ```
