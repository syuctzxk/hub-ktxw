# 待选方案1
 - 步骤1：先用dots.ocr工具将pdf文件解析为markdown格式，pdf的每一页内容被转换成1个md文件
 
 - 步骤2：分别遍历每个pdf解析成的子md文件，并将其按页码把内容拼接在一起，保存为一个excel文件
 
 - 步骤3：将原始的带有公式的md文件内容和excel文件中的内容进行合并
 
 - 步骤4：用qwen3-Embedding模型对合并后的内容和用户的问题分别进行编码
 
 - 步骤5：计算每个问题与公式之间的相似度，保留8个最相似的公式
 
 - 步骤6：将问题与这8个公式同时输送给qwen的think模型，要求模型给出最终的结果
 
# 待选方案2
 - 步骤1：使用PyPDFLoader直接提取pdf的内容，直接读取markdown文件的内容，将内容保存为csv文件，每个文件的内容为1行
 
 - 步骤2:基于文本匹配得到每个问题对应的公式以及其对应知识
 
 - 步骤3 分别将问题和所匹配的公式对应的知识输入给大模型，要求大模型输出最终结果
 
# 待选方案3
 - 步骤1：使用pyMupdf提取pdf的内容，直接读取markdown文件的内容，并将内容保存
 
 - 步骤2：用向量分别对公式内容和提问进行编码，然后通过相似度计算与提问最匹配的3个公式文档
  
 - 步骤3：将匹配的3个公式文档与提问输入给大模型，要求大模型给出其选择的公式，公式所需要的参数，以及最终答案
 
 - 步骤4：根据大模型所给的公式和参数用sympy进行精确计算
