## 三、宏观经济数据类数据源
### 1. 中国国家统计局

## 四、金融股票数据类数据源
### 1. AkShare库
- 文件 : stock_akshare.py
- 数据源 : AkShare是一个开源的Python金融数据接口库
- 用途 : 获取股票财务摘要、三大财务报告（资产负债表、利润表、现金流量表）等金融数据
- 数据类型 : 上市公司财务指标、财务报表数据等
### 2. 百度股市通
- 文件 : stock_bd.py
- 数据源URL : http://finance.pae.baidu.com
- 用途 : 获取股票/板块搜索、K线数据、三大会计报表、企业十大股东、主营业务结构、高管信息等
- 数据类型 : 股票基本信息、财务数据、股东信息等
### 3. 香港交易所
- 文件 : report_hk.py
- 数据源URL : https://www1.hkexnews.hk
- 用途 : 获取香港上市公司的公开报告，如年报等
- 数据类型 : 上市公司公告、年报等PDF文件
### 4. 上海证券交易所
- 文件 : report_sh.py
- 数据源URL : https://query.sse.com.cn
- 用途 : 获取上海证券交易所上市公司的公告和报告
- 数据类型 : 上市公司公告、年报、业绩预测等
### 5. 深圳证券交易所
- 文件 : report_sz.py
- 数据源URL : https://www.szse.cn
- 用途 : 获取深圳证券交易所上市公司的公告和报告
