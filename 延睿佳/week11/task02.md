## 作业2：文本 vs 文本+图 提问的处理区别

| 项目 | 文本提问 | 文本 + 图提问 |
|------|-----------|----------------|
| **输入模态** | 单一模态（自然语言） | 双模态（语言 + 视觉） |
| **特征提取** | 使用文本编码模型（如 BGE, E5）提取向量 | 使用多模态编码模型（如 CLIP, Qwen-VL）对文本与图像进行联合编码 |
| **检索方式** | 基于文本嵌入在 Milvus 中搜索 | 同时计算文本嵌入与图像嵌入，进行多模态联合相似度检索（可采用加权融合或 cross-modal matching） |
| **生成阶段** | RAG模型仅根据文本上下文生成答案 | 生成模型（如 Qwen-VL-Chat）同时解析图像内容并结合文本上下文回答问题 |
| **应用示例** | “请介绍一下悬索桥的结构原理” | “请结合这张图解释悬索桥的主要承力结构” |
| **技术重点** | 文本语义匹配与知识检索 | 视觉语义对齐、跨模态检索、联合注意力机制 |

### 总结：
- 文本提问 → 检索与生成完全在语言空间；
- 文本 + 图提问 → 同时利用视觉与语言特征，需要多模态融合模型（如 CLIP + LLM 或 BLIP2）。
